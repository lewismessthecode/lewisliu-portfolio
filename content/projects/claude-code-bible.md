A working reference for getting real production work out of Claude Code:

- Curated agent skills covering planning, debugging, code review, and
  domain workflows.
- Prompt templates and slash commands designed around the rhythm of
  day-to-day engineering.
- Notes on context window economics, model selection (Opus / Sonnet /
  Haiku), and when to delegate to sub-agents.
- Hooks, settings, and patterns that make Claude Code feel less like
  a chat box and more like a teammate.

Built from the patterns I actually use, then sanded down until other
people can pick them up cold.
