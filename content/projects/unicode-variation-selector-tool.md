An educational web app demonstrating Unicode steganography inspired by Karpathy:
- Encode and decode secret messages hidden within emojis using Unicode variation selectors
- Clipboard copy and Web Share API integration
- Responsive design for desktop and mobile
- Local storage persistence for encoded messages
- Educational demonstration of Unicode character manipulation
