Most chat UIs hide the model's chain of thought. This one foregrounds it:

- Streaming UI separates "thinking" from "answer" so you can watch the
  model reason in real time.
- Clean typography, fast input, no chrome — built for sustained back-and-forth.
- Drop-in DeepSeek API client with sensible defaults.
- Designed as a personal sandbox for studying reasoning models, not as a
  full chat product.
