A local-first bioinformatics workflow platform with agentic execution:
- FastAPI backend with SQLite database and SSE event streaming
- Agent orchestration for automated workflow execution
- Nextflow and MiniWDL workflow adapters
- Next.js frontend with real-time event consumption
- Docker support for containerized deployment
- Async SQLAlchemy with Alembic migrations
